📌 Project Overview

This project explores how AI tools can be incorporated into the Business Intelligence workflow to improve productivity while building dashboards in Power BI.

The objective was to combine SQL, AI-assisted analysis, and Power BI to create an interactive banking analytics dashboard.

🎯 Objectives
Explore AI-assisted data analysis workflows
Practice SQL data generation and transformation
Build an interactive Power BI dashboard
Learn how AI can support KPI and DAX ideation
Strengthen data cleaning and visualization skills

🔄 Project Workflow
Data Preparation
Generated sample banking data using SQL and Perplexity
Identified and corrected inconsistencies in date columns
Combined three related tables using SQL JOINs
Power BI Development
Imported data into Power BI
Cleaned and transformed data
Designed interactive dashboards
Implemented AI-assisted KPI and DAX recommendations
Applied formatting and report enhancements

📊 Dashboard Features
Number of Transactions by Type
Transactions by Month
Transactions by Customer
Total Balance by Account Type
Inactive Accounts by Year and Month
Customer Count by Gender
Customers by Age Group
Treemap Visualization
Additional interactive charts and KPIs

🛠️ Technologies Used
SQL
Power BI Desktop
DAX
Power Query
Perplexity AI

📚 Key Learnings
Using AI as a productivity tool in BI development
SQL joins and data preparation
Data cleaning and transformation
Dashboard design and visualization
Creating KPIs and DAX measures
Building interactive business reports

⚠️ Note
This project was developed for learning and practice purposes. The dataset was generated for educational use and does not contain real customer or business information.
